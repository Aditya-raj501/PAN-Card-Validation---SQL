# PAN-Card-Validation---SQL
This project demonstrates an end-to-end data cleaning and validation pipeline for Indian Permanent Account Number (PAN) records using SQL.  The objective is to clean raw PAN data, validate each PAN according to business rules, classify records as Valid or Invalid, and generate a summary report.

Business Problem

Organizations frequently receive PAN data from multiple sources. These datasets may contain:

Missing values
Duplicate records
Incorrect formatting
Invalid PAN structures
Lowercase characters
Extra spaces
Invalid sequences
Special characters

Poor data quality affects downstream analytics and compliance.

This project automates the validation process.
